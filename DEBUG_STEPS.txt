Server đang chạy tại: http://localhost:5173

════════════════════════════════════════════════
BƯỚC 1: Kiểm tra browser
════════════════════════════════════════════════

1. Mở browser (Chrome/Safari/Firefox)
2. Truy cập: http://localhost:5173
3. Nhấn F12 (mở DevTools)
4. Tab Console
5. Xem có dòng màu ĐỎ nào không?

Nếu CÓ error → Copy toàn bộ error gửi lại

════════════════════════════════════════════════
BƯỚC 2: Check Network
════════════════════════════════════════════════

1. Tab Network trong DevTools
2. Reload page (Cmd+R hoặc Ctrl+R)
3. Xem có request nào màu ĐỎ không?
4. Có request tới `supabase` fail không?

════════════════════════════════════════════════
BƯỚC 3: Test nhanh
════════════════════════════════════════════════

Try hard refresh:
- Mac: Cmd + Shift + R
- Windows: Ctrl + Shift + R

════════════════════════════════════════════════
BƯỚC 4: Chạy command này
════════════════════════════════════════════════

Open terminal trong dự án và chạy:

curl http://localhost:5173

Nếu thấy HTML → server OK
Nếu error → server có vấn đề
