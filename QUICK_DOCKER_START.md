# 🚀 Quick Start - Docker

## ⚡ Chạy nhanh:

```bash
# 1. Copy .env nếu chưa có
cp .env.example .env

# 2. Điền Supabase credentials vào .env

# 3. Build và chạy
docker-compose up -d --build

# 4. Truy cập
# http://localhost:3000
```

## 📋 Commands:

```bash
# Stop
docker-compose down

# Rebuild
docker-compose up -d --build

# Logs
docker-compose logs -f
```
