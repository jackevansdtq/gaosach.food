CÁCH 1: Personal Access Token
───────────────────────────────────────────────────────

1. Tạo token: https://github.com/settings/tokens
2. Chạy lệnh:
   export GITHUB_TOKEN=your_token_here
   git remote set-url origin https://${GITHUB_TOKEN}@github.com/jackevansdtq/gaosach.food.git
   git push -u origin main

CÁCH 2: GitHub CLI
───────────────────────────────────────────────────────

brew install gh
gh auth login
git push -u origin main

CÁCH 3: SSH Key
───────────────────────────────────────────────────────

cat ~/.ssh/id_rsa.pub
# Add to GitHub: https://github.com/settings/keys

git remote set-url origin git@github.com:jackevansdtq/gaosach.food.git
git push -u origin main
